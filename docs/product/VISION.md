# Project Etna - Product Vision

## Mission

Project Etna is an **AI-powered Silicon Debug Assistant** designed to help verification engineers debug RTL designs and analyze hardware behavior more efficiently.

## Target Users

| User Segment | Description | Key Needs |
|--------------|-------------|-----------|
| **Verification Engineers** | Professional ASIC/FPGA engineers | RTL analysis, waveform debugging, root cause analysis |
| **Students** | EE/CS students learning digital design | Learning tool, affordable access |
| **Hobbyist FPGA Developers** | Makers, open-source hardware contributors | Accessible tooling without expensive EDA licenses |

## Core Value Proposition

1. **AI-Assisted Debugging** - LLM-powered analysis of RTL code, identifying bugs, race conditions, and timing issues
2. **Unified Debug Environment** - Single interface for RTL code + waveforms + AI chat
3. **Accessible** - Web-based, no expensive EDA tool installation required
4. **Modern UX** - Snappy, responsive interface unlike legacy EDA tools

## Competitive Landscape

| Tool | Type | Cost | AI-Powered |
|------|------|------|------------|
| **Cadence Verisium** | Enterprise | $$$$ | Yes |
| **Synopsys Verdi** | Enterprise | $$$$ | Limited |
| **GTKWave** | Open Source | Free | No |
| **Surfer** | Open Source | Free | No |
| **Project Etna** | SaaS | Freemium | Yes |

## Differentiators

- **AI-native**: Built from ground up with LLM integration
- **Web-first**: No installation, works in browser
- **Affordable**: Freemium model accessible to students/hobbyists
- **Modern stack**: Next.js, real-time streaming, responsive design
